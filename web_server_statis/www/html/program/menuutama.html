Aplikasi inix, merupakan aplikasi yang menerapkan Bootstrap dengan fitur web mobile. yang merupakan contoh aplikasi untuk memfasilitasi transaksi menambah daftar buku, edit buku, menghapus data, dan menampilkan daftar buku. aplikasi ini, menggunakan Fasilitas Shelf yang di miliki dart untuk membuat web server dan di hubungkan dengan database MySQL(MariaDB).
